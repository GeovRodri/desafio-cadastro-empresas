# Desafio Cadastro de Empresas

## Tecnologias:
* Angular
* NodeJs (Utilizando JavaScript)
* MongoDB
* Swagger

## Serviços utilizados para colodar em produção
* Firebase Hosting
* Cloud Functions
* MongoDB Atlas

## Para rodar o projeto:
* Angular: cd front-end/ && npm run start
* NodeJs: cd back-end && npm run start
